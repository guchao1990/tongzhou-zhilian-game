## 一、执行摘要

**公司名称**：同舟智联科技（深圳）有限公司

**创立背景**：2026年，全球物联网市场持续高速增长，工业数智化转型加速推进。公司创始团队具备物联网、嵌入式系统、云计算等领域深厚积累，决定以"工业智能网关"为核心产品切入市场，提供软硬件一体化解决方案。

**当前资金**：启动资金1万元人民币
**投资机制**：后续资金投入与公司盈利情况挂钩，实行分阶段注资

**核心理念**：轻资产启动，敏捷验证，小步快跑。用最小成本验证市场，快速迭代确保方向正确。

## 二、当前宏观环境分析（投资顾问视角）

### 2.1 宏观经济环境

**全球经济增速放缓，企业端数字化投入意愿分化**
- 2025-2026年全球经济仍处于调整期，企业普遍收紧资本开支
- 但数字化转型投入逆势增长，企业希望通过数智化降本增效
- 工业领域：智能制造2025年市场规模突破3.2万亿元，年增速12%+

**政策利好持续**
- "十四五"智能制造发展规划持续推进
- 新型工业化战略加大支持力度
- 专精特新企业扶持政策密集出台

**资本市场环境**
- 一级市场投资趋于理性，更关注商业化落地能力
- 硬科技赛道仍受青睐，特别是自主可控方向
- 创业企业需具备"自我造血"能力，降低对外部融资依赖

### 2.2 行业赛道判断

| 赛道 | 市场规模 | 增速 | 竞争格局 | 创业公司机会 |
|-----|---------|------|---------|------------|
| 工业IoT网关 | 800亿+ | 15%/年 | 头部分散 | 中小场景切入 |
| 环境监测终端 | 300亿+ | 20%/年 | 高度分散 | 性价比产品 |
| AIoT云平台 | 千亿级 | 25%/年 | 头部垄断 | 垂直行业PaaS |

**核心判断**：工业IoT网关是目前最优切入点——市场够大、增速够快、竞争相对分散、客户付费意愿强。

### 2.3 投资建议

**当前环境下的生存策略**：
1. **轻资产运营**：控制固定成本，优先外包非核心环节
2. **小步快跑**：用最小MVP验证市场，避免重投入押注
3. **客户导向**：先收钱再开发，用订单驱动研发
4. **分阶段注资**：根据盈利情况决定后续投入节奏

## 三、公司主营业务确认

### 3.1 核心产品线（产品经理 × 架构师联合确认）

**一号产品：同舟智联·工业智能网关**（核心收入来源）

产品定位：面向工业制造、楼宇设备、智慧农业等场景的边缘计算网关

核心功能：多协议接入（Modbus/OPC-UA/PLC等10+种）、边缘计算、断网续传、远程运维

目标客户痛点：设备孤岛、运维成本高、进口设备替代需求

定价：硬件2000-4000元/台，SaaS订阅200-400元/台/年

**二号产品：同舟智联·智能环境监测终端**（现金流补充）

产品定位：高性价比多参数环境监测终端

核心功能：温湿度/CO2/光照等多参数、LoRa无线、电池续航2年+

目标客户：大棚种植户、冷链仓储、数据机房

定价：硬件500-1200元/套，SaaS 100-200元/套/年

**三号产品：AIoT云平台**（长期壁垒构建）

产品定位：面向设备制造商的物联网云平台

核心功能：设备接入、数据管理、规则引擎、远程运维

定价：基础版免费、专业版2000元/年、企业版2万元/年

### 3.2 主营业务确认

**确认：同舟智联以工业智能网关为核心产品，提供"硬件+平台+服务"一体化解决方案**

差异化定位：
- 比大厂更专注工业场景的垂直定制
- 比小厂有更强的云平台能力和服务品质
- 以网关产品为入口，后续叠加传感器、云服务形成持续收入

## 四、商业模式与盈利路径

### 4.1 商业模式画布

**价值主张**：为工业客户提供可信赖、高性价比的物联网边缘计算解决方案，帮助客户实现设备数字化、运维远程化

**核心客户**：中小型工厂设备商、楼宇设备制造商、农业合作社

**核心资源**：技术团队（嵌入式+云平台）、客户关系、行业积累

**关键活动**：产品研发、客户服务、技术支持

**渠道触达**：直销（优先）+ 经销商渠道 + 行业展会

**收入来源**：硬件销售（主）+ SaaS订阅（增）+ 项目定制（附加）

**成本结构**：研发人力（主）+ 硬件BOM（次）+ 云服务成本 + 市场推广

### 4.2 盈利路径

**第一阶段（0-12个月）：生存验证期**
- 用1万元启动资金完成产品原型
- 目标：获得首批10-20个种子客户，实现30-50台出货
- 关键：活下来，验证产品有人买单

**第二阶段（12-24个月）：自我造血期**
- 用第一阶段收入支撑运营
- 目标：月出货100台+，月收入20万+，盈亏平衡
- 关键：跑通商业模式，具备自我造血能力

**第三阶段（24-36个月）：规模化期**
- 根据盈利情况决定追加投入
- 目标：月出货500台+，年营收500万+
- 关键：建立渠道网络，形成规模效应

### 4.3 投入与产出测算

**启动阶段（0-6个月）**

| 项目 | 金额 | 说明 |
|-----|------|------|
| 启动资金 | 10,000元 | CEO投入 |
| 原型开发 | 5,000元 | 样品打样、少量物料 |
| 云平台初期 | 3,000元 | 最基础的云服务器 |
| 营销推广 | 2,000元 | 线上推广、样品寄送 |
| 合计 | 10,000元 |  |

**里程碑目标**
- 第3个月：完成原型机
- 第6个月：签约首批10个种子客户
- 第12个月：月出货50台，月收入10万+

**盈亏平衡测算**
- 固定成本（研发1人 + 运维）：约2万/月
- 边际成本（每台网关BOM+云服务）：约1000元/台
- 定价：硬件2500元/台 + SaaS 300元/台/年
- 盈亏平衡点：月出货约60台

## 五、分阶段投资机制

### 5.1 投资原则

1. **先投入后增资**：每一分投入都要对应明确产出
2. **与盈利挂钩**：公司盈利是后续增资的前提条件
3. **风险共担**：创始团队与投资人共担风险
4. **透明管理**：资金使用情况定期公示

### 5.2 分阶段注资计划

**第一阶段：启动期（已完成）**

- 投资方：CEO顾超个人出资
- 投资金额：1万元
- 资金用途：原型开发、云平台搭建、首批客户验证
- 股权比例：待明确（建议CEO占股80%，团队预留20%）

**第二阶段：生存期（12个月内）**

- 触发条件：公司月出货超50台，且累计收入超20万
- 增资额度：5-10万元
- 资金来源：CEO追加 / 利润滚动 / 战略投资人
- 资金用途：扩充团队（嵌入式×1）、量产备货
- 对赌条件：12个月实现盈亏平衡

**第三阶段：成长期（12-24个月）**

- 触发条件：第二阶段目标达成，且具备规模化能力
- 增资额度：20-50万元
- 资金来源：外部融资 / 战略合作方
- 资金用途：扩充销售团队、扩充产品线、扩大产能

**第四阶段：规模化期（24个月后）**

- 触发条件：年营收突破500万
- 增资额度：根据发展需要
- 资金来源：机构融资 / 银行贷款 / 战略投资

### 5.3 增资决策流程

1. CEO提出增资申请，说明用途和预期回报
2. 管理层会议审议（CEO + CTO + 产品经理）
3. 达到约定里程碑后启动增资
4. 资金到账后公示用途和使用情况

## 八、总结与展望

**创业初心**：用技术真正帮助中小企业实现设备智能化，降低运维成本，提升生产效率。

**短期目标**：用1万元启动资金验证产品有人买单，在6个月内实现从0到1的突破。

**中期目标**：用12-24个月实现自我造血，形成可持续的商业模式。

**长期愿景**：成为工业物联网领域的专业品牌，为中国智造贡献一份力量。

**核心理念**：
- 轻资产启动，不烧钱
- 客户导向，先收钱再开发
- 小步快跑，快速迭代
- 与公司盈利挂钩，稳健发展

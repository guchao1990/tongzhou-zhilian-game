## 3. 数据库设计

### 3.1 下注规则

**底注**：每局开始时，所有玩家投入相同底注（ANTE = 50），形成初始底池。

**明注与暗注**：
- **暗注**：未看牌状态下跟注/加注，金额为当前明注的 2 倍
- **明注**：已看牌状态下跟注/加注，金额等于当前明注

**看牌**：玩家可花费暗注金额翻开自己的手牌，看牌后后续下注按明注计算。

**加注**：在明注基础上额外投入，加注后视为已看牌，明注和暗注同时更新。

### 3.2 轮次规则

- 玩家按顺时针顺序依次行动（庄家 → 下家 → 下家 → …）
- 每位玩家每轮都必须做出选择：**跟注**、**看牌**、**加注** 或 **弃牌**
- 弃牌玩家跳过，不再参与本局后续轮次
- 从庄家开始，每位活跃玩家都表态后，本轮结束，进入下一轮
- 剩余玩家降至 **2 人**时，自动进入开牌阶段

### 3.3 胜负判定

**开牌条件**：剩余 2 名活跃玩家时，强制开牌比大小。

**牌型大小**：炸弹 > 金花 > 顺子 > 对子 > 单张（同牌型比点数，对子特殊处理）

**结算**：牌型最大者赢得底池。
